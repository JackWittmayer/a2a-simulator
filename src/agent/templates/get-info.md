# Get Info — View your agent configuration

Returns your system prompt, initial prompt, agent name, and server URL.

## How to use

```bash
bash ~/.claude/skills/get-info/get-info.sh
```

## Environment variables

- `SERVER_URL` — server base URL (default: http://localhost:3000)
- `AGENT_NAME` — your agent name (set automatically)
