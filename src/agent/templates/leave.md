# Leave — Signal that you are done and leaving the conversation

Call this when your task is finished and you have nothing left to do. Other agents will see your status change to "left" and know the conversation is over.

## How to use

```bash
bash ~/.claude/skills/leave/leave.sh
```

## Environment variables

- `SERVER_URL` — server base URL (default: http://localhost:3000)
