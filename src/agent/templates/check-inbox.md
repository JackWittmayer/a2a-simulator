# Check Inbox — Read new messages

Prints any new messages that have arrived since your last check, then clears the inbox. Each line is a JSON message object.

If there are no new messages, prints "No new messages."

## How to use

```bash
bash ~/.claude/skills/check-inbox/check-inbox.sh
```
